Herramienta de búsqueda inteligente que localiza y extrae "trozos" (snippets) de archivos de texto embebidos dentro de binarios.

- Busca automáticamente archivos `.xml`, `.ula`, `.lua`, `.json` y `.txt`.
- Muestra el **Offset Hexadecimal** exacto del hallazgo.
- Imprime una previsualización (snippet) del contenido de 20 bytes antes y después del patrón.
- Fundamental para localizar tablas de armas de un juego.

### Uso

```bash
python pattern_extractor.py <archivo.bin>
```
